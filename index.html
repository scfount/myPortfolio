NASA Feature Website

Sep 2019 - HTML / CSS / JS / API / Vue

Designed and developed a responsive multi-page site utilizing Axios with NASAs REST API's, integrating a search feature, within the front-end framework Vue.

NASA Feature Website - View here

This was my first project incorporating APIs and using Vue.js as a front-end framework and bootstrap for style. I decided to work with APIs from NASA as there was plenty of great documentation to get up and running and very interesting content. The First API is used was the NASA Astrological image of the day, which updates daily with a new astrological image or video, title, and description which I incorporated into my home page. One challenge I faced was ensuring that my vue component would generate the correct media tag as I would not always get an img from the API, it could also be a video. Using a v-if-else statement I was able to conditionally render the appropriate media tag.

The second API I used pulls all the tech projects NASA is working on. I initially pull the projects updated within the last month and used a vue component to generate a tile with the project ID and last update date. From there, the user can type the project ID they would like to learn about into the search box, re-running the API query with project ID parameters. The project is displayed on the page with a title, description, benefits, start and expected end date, and whether it is still an active project or not. Some challenges I faced were only displaying projects that were updated within the last month as the original query returned thousands of projects. Dog Adoption Website

June 2019 - HTML / CSS / JS / jQuery

Developed a responsive multi-page website based on provided specifications utilizing JS and jQuery to create an 'add to cart' feature.

Dog Adoption Website - View here

For this project I was given several page mockups and went through the process of converting them into a fully responsive and functional website. Initially I broke down the pages into semantic HTML sections, coding all the markup first. Once all of the pages were done, I added my style using current standards such as grid and flexbox for ease of layout and responsiveness.

There were several added features for this project, the first was adding an alert that prompted the user with information about the dog. For that I wrote a JavaScript function with three parameters â€“ name, breed and fee. The were passed as arguments when the user clicked on the dog tile.

The second feature was a functioning cart total, so when a user added a dog to their cart it would update with the current total. This was achieved using JavaScript and jQuery. When the add to cart button is clicked, this calls the function addToCart, which accepts the fee as an argument, and updated the cartTotal variable. This variable is trimmed to two decimal places and placed next to the shopping cart icon using jQuery.

The third feature was generating all the blog posts dynamically utilizing a for loop to iterate through an array of blog posts, selecting and manipulating DOM elements with JavaScript.
